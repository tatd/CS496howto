Starting with HTML

Before we connect to the API, let's start out with a basic form that we can work off of. In your index HTML document, place this between the <body> tags.

<!-- app search page-->
<div data-role="page" id="pokeSearch-page">
    <div data-role="header" data-theme="b">
         <h1>Pokeapi Demos</h1>
    </div>
    <div data-role="main" class="ui-content">
    <form id="pokeForm" action="#pokeDetails-page" method="post">
        <div>
        <label for="pokeInput">Pokemon name or number:
        <input type="text" name="pokeInput" id="pokeInput" placeholder="Name/Number" value="">
        </div>
        <br>
        <div>
        <a class="ui-btn" onclick="pokeSubmit();">Search
        </div>
    </form>
    <div>
    <ul data-role="listview" data-inset="true" id="pokeDetails">
    </ul>
    </div>
    </div>
</div>

This will be the code for our single-page application. The main thing here is a form to get the user input. Instead of a traditional submit button, we just have a normal button with an onClick function to handle the interaction with the API. Also, beneath the form, I've included a div that will display the data we recieve from the API. jQuery Mobile uses ul data-role="listview" as a list element with more features.

Pokeapi Guide

By Dennis Tat


This is a guide on how to use Pokeapi, a web API that contains data on the Pokemon video games. From Pokeapi, you can get information on over 700 Pokemon such as their names, descriptions, types, and even images of the sprites used in the video games. Using this information, you could build your own web-based Pokedex (which is basically an encyclopedia of Pokemon).


Pokeapi accepts HTTP requests via GET. This API does not require an API key, so anyone can immediately jump in and start making requests. However, there is a daily limit of 300 requests for each resource (so each specific URL can be called up to 300 times per day).


This guide will walk through how to start making requests to Pokeapi using JavaScript and go through some sample calls so that you can start constructing your own Pokedex.
